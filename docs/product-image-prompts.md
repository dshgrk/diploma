# Prompts For Realistic Product Images

Этот файл хранит готовые промпты для ChatGPT Image Generation под все `40` товара витрины.

## Общие правила

- Все промпты ниже рассчитаны на `realistic luxury ecommerce product photography`.
- Для генерации использовать `transparent background`.
- Основной файл для товара подписывать строго по `slug`:
  - `quiet-pearl-ring.png`
  - `moon-bracelet.png`
  - `white-diamond-earrings.png`
- Если нужны дополнительные ракурсы, использовать:
  - `<slug>-detail-1.png`
  - `<slug>-detail-2.png`
  - `<slug>-hover.png`
- Если генератор добавляет лишнее, дописывать в конец:
  `no model, no hand, no neck, no box, no props, no text, no logo, no watermark, no background scene`.
- Если нужно больше соответствия карточке товара, просить `front-facing centered composition` и `accurate jewelry proportions`.
