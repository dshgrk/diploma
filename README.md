# Веб-система для продажу авторських прикрас з конструктором дизайну

## Опис проєкту
Даний проєкт являє собою веб-систему для продажу авторських прикрас із підтримкою повної персоналізації виробів. Основною функціональною складовою системи є інтерактивний конструктор дизайну, який дозволяє користувачам самостійно формувати зовнішній вигляд прикраси шляхом вибору доступних параметрів та миттєво переглядати результат у візуальному форматі.

Система забезпечує автоматичний розрахунок вартості виробу на основі обраних характеристик, що дозволяє оформлювати замовлення без участі продавця-консультанта та зменшує ймовірність помилок у процесі узгодження деталей.

Проєкт орієнтований на використання у сфері виготовлення та реалізації індивідуальних авторських прикрас і може застосовуватись у невеликих майстернях або приватних брендах.

## Основні функціональні можливості
- перегляд сайту-вітрини та каталогу готових прикрас;
- створення індивідуального дизайну прикраси за допомогою конструктора;
- інтерактивна візуалізація змін параметрів прикраси;
- автоматичний розрахунок підсумкової вартості виробу;
- оформлення онлайн-замовлення;
- збереження та обробка замовлень у системі;
- панель адміністратора для перегляду та зміни статусів замовлень;
- керування товарами, матеріалами та цінами.

## Стек технологій
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Version Control:** Git, GitHub

## Статус проєкту
Проєкт перебуває на етапі планування та проєктування. Функціональність реалізується поетапно відповідно до обраної гнучкої методології розробки.
